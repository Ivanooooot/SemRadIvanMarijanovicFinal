# Seminarski Rad - Marijanovic

This is a web application project for selecting a vehicle in a rent a car company in order to book a reservation. 
Based on the proposed categories of vehicles, the customer selects one that includes a pair of desired vehicle models.

## About

The purpose of the project is to create a web application for a rent a car company.

## Getting Started

From the platform https://github.com/Ivanooooot/SemRadIvanMarijanovicFinal, download all the files to start the project on your computer.

## Installing

To start this project, the necessary programs are:
- Visual studio
- database browser

## Usage

After starting the visual studio program, in the appsettings.development.json folder it is necessary to adjust the connection string to your local computer and database.

For testing Role Manager & User roles: login as Administrator
